# Governance

NemoClaw Community is maintained by the NVIDIA NemoClaw maintainers.

## Maintainer Responsibilities

- Review and merge pull requests.
- Keep public documentation accurate and safe to follow.
- Maintain license headers, third-party notices, and dependency records.
- Enforce DCO sign-off requirements.
- Route security reports through NVIDIA's private vulnerability disclosure process.
- Complete NVIDIA internal open-source review before public release, major scope changes, or new dependency distribution.

## Change Management

Pull requests should be small enough to review directly. Changes that add dependencies, alter distribution scope, publish container images, or materially change the project license/compliance surface require maintainer review before merge and may require renewed internal open-source review.

## Releases

Public releases should match the scope approved by NVIDIA's internal open-source review. If the project adds new packages, changes package licenses, changes distribution model, or starts publishing container images, maintainers should revisit the approval scope before release.
