# Maintainers

NemoClaw Community is maintained by the NVIDIA NemoClaw maintainers.

## Code Owners

Repository ownership is configured in [.github/CODEOWNERS](.github/CODEOWNERS).

## Responsibilities

- Review pull requests for correctness, security, and release-scope impact.
- Keep third-party notices and license headers current.
- Keep example setup documentation reproducible.
- Coordinate NVIDIA internal open-source review before public release or material scope changes.

## Security

Security reports should follow [SECURITY.md](SECURITY.md), not public issues.
